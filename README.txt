DOZERX — ADD YOUR MACHINE PHOTOS HERE
======================================

Put your photos in this folder with these exact filenames:

  excavator.jpg   → shows on Excavator card
  bulldozer.jpg   → shows on Bulldozer card
  crane.jpg       → shows on Tower Crane card
  tipper.jpg      → shows on Tipper Truck card
  compactor.jpg   → shows on Compactor card
  grader.jpg      → shows on Motor Grader card

Photo tips:
- Landscape orientation (16:9 ratio works best)
- Minimum 800px wide
- The site auto-darkens images for readability
- If a photo is missing, the card shows a placeholder until you add it

Each machine card on index.html links to rent.html?type=MACHINENAME
Each listing on rent.html has a "Request this machine" button.

To add real listings, edit the `machines` array in rent.html (line ~60).
